Instructions

Search and visualize default keyboard shortcuts for Ableton Live 8 and 9.

For Mac keyboards with or without a numeric keypad, and for generic PC keyboards.

How to use:

  • Click on a category on the left, then hover over a function on the right.
  • Alternatively, enter a keyword into the search input field, and hover over the results.
  • The appropriate keyboard shortcut will light up.
  • Commit to memory!

Tooltips

Some Ableton Live functions have tooltips, which can be activated by hovering over them with the mouse.

Tooltips are used:

  • whenever there's additional information about a function
  • whenever there's an alternative keyboard shortcut for a function
  • whenever invoking an Ableton Live function involves a mouse-click

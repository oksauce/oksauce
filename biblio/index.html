b2 app

brennan b2 is a fantastic piece of kit, with great software support, however, on a busy family network, static polling every half a second, seemed to frequently get in the way of uploadng music & sometimes listening to the radio, the client software also requires an internet connection

> sauce software upgrades your brennan b2 with :

  • new and modified javascript
  • performance & style updates
  • freed up WIFI or wired network
    from static polling sync traffic
  • respecting your privacy, the b2
    no longer requires an internet
    connection to play stored music
  • mobile & laptop event driven sync,
    syncs with the brennan b2 music server
    only when needed
  • super fast music files upload
    [ no sync traffic conflicts ]
  • zero b2 music player upload issues
  • multiple devices access b2 server
    without sync network traffic overhead
  • improved performance & user experience
brennan b2 music player is based on open source hardware with application software distributed via brennan.co.uk providing advice for b2 users to customise [ upgrade ] hardware & software, and use any version of brennan b2 software available from brennan.co.uk or and from the open source community, available here :
  • https://github.com/geometryzen/
    brennan-app (sponsored by brennan.co.uk jan 2019)
  • https://github.com/aromis/b2

brennan b2 recoded > sauce software, implements * lazy event driven sync, updated track time control, tone & bass controls & new track countdown feature

* lazy master slave relationship, between all brennan b2 network client devices, slave can become master, master can become slave, for maximum usability
